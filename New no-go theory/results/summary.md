Tests the candidate prediction that the tangent-vector difference x_S = d(rho_full)/dtheta - d(rho_cut)/dtheta obeys ||x_S|| ~ Gamma^-nu, and that the sector-mediated quantum Fisher information F_Q,S = x_S^dagger G_rho x_S ~ Gamma^-2nu, translating the three-tier response classification into a three-tier metrological classification.

The rank-1 (non-Hermitian, Sherman-Morrison) sector cut gives the difference x_S the fixed-direction-times-scalar structure needed for the *whole vector* to vanish together at a single real control value, confirming the vector generalization of the hidden class transition exactly, within the abstract scope of Theorem III (no Hermiticity assumed there).

## Gate M3: nu -> 2nu QFI translation (physical 3-level Lindblad model)

Overall: **PASS**

- nu[x_S] (direct fit) = 1.9962
- nu[F_Q,S] (direct fit, should be ~2*nu[x_S]) = 3.9954

This is the core numerical confirmation of the user's prediction: in a genuine 3-level Lindblad master equation with a Gamma-scaled singular dissipator, the sector-mediated QFI decays exactly twice as fast (in the Gamma^-nu sense) as the tangent-vector norm itself.

## Gate M4: physical (Hermitian-coupling) interior transition search

Interior transition found: **False**

- generic-point norm: 2.121e-12
- best found (2D search) norm: 7.071e-13
- suppression ratio vs. generic: 3.00

No interior (non-trivial) exact or near-exact vector transition was found with a 2-real-parameter (amplitude, phase) physical Hermitian coupling cut; the leading-order response x_{S,0} lives in the 3-real-dimensional traceless-Hermitian slice of the protected block, so codimension counting predicts an interior zero generically requires 3 independent real controls. This is consistent with (not a failure of) the theory: it correctly predicts when a hidden transition is and is not achievable with a given control budget. The exact vector transition IS realized with a rank-1 (non-Hermitian) abstract cut in Gate M2.

## Gate M5: rank-deficiency negative control

Overall: **PASS**

As epsilon -> 0 the steady state approaches a near-pure (rank-deficient) state, the SLD metric approaches singularity, and the clean nu_F = 2*nu_x relation degrades -- confirming that the 'full rank / non-singular SLD metric' assumption in the prediction is load-bearing, not decorative.

## D1: oblique (Riesz) protection vs the old orthogonal-projector formula

Overall: **PASS**

- Hermitian control (`a=0`): Riesz and orthogonal projector formulas agree
  exactly (diff = 0.0), as required.
- False-suppression point: at `a* = 1.1488` the old orthogonal formula gives
  `r0_orth ~ 1.2e-16` (predicts suppression / Class II), while the true
  Riesz coefficient `r0_riesz = 6.239` and the measured plateau at
  `Gamma=1e7` agree to relative error `3.1e-7` -- the response is in fact
  protected (Class III); the old formula misclassifies it.
- At `a=1,2,5` the orthogonal formula is wrong by factors of 12.7x, 29.9x,
  305x respectively (relative error vs. the measured plateau), while the
  Riesz formula matches to <1e-6 in every case.
- Dissipation-enhanced (non-monotonic) sector response: at `a=4` (with
  `dQ1=0.05, dQ2=3.0`), `|R_S(Gamma)|` has an interior maximum (`Gamma~6.0`,
  peak value 178.6) roughly 200x above both the small-`Gamma` and large-`Gamma`
  (plateau, 0.87) values -- confirmed not a numerical artefact of a near-
  singular matrix (`cond(A)` grows smoothly through the bump). The `a=0`
  Hermitian control shows no such interior maximum over the same window.

## D2: cancellation-promoted suppression and the symmetry-free exact zero

Overall: **PASS**

- Order promotion: with the 2-hop parallel-path coupling tuned to
  `v23* = -0.7846` (near-cancellation of the master moment `m_2`), the
  measured suppression order is `nu = 4.0004`, versus the naive per-kernel
  (single shortest path, 2-hop) graph-distance bound of `nu = 3`. Detuning
  `v23` away from the cancellation point by `+0.1` demotes the response back
  to the naive order (`nu = 3.0004`), confirming the promotion is due to
  interference, not a hidden exact zero.
- Exact symbolic Class I with no reducing symmetry: for the rank-1 cut
  `M_full = M_cut + t*(u v^T)` with `v` chosen (exact rational arithmetic)
  orthogonal to `{c, M_cut c}`, all three augmented Krylov moments vanish
  identically **for every t** (verified symbolically: `[0, 0, 0]`), certifying
  an exact all-order Class I master response by Theorem 4.2 / Cayley-Hamilton.
  At the same time the cross-kernel `K01` (the abstract analogue of the
  coherent-Raman vertex `K12`) is nonzero and different between full and cut
  (`K01_full = -0.4577`, `K01_cut = 0.3059`), and no nontrivial permutation
  symmetry commutes with `M_full`. The old scalar criterion
  (`K12*K21 = 0`) and the old symmetry audit both miss this exact zero.

## D3: the class belongs to the scaling path, not the material

Overall: **PASS**

- Calibration: the two decompositions (`Gamma*D_T + B_T` and
  `Gamma*D_U + B_U`) agree exactly at the reference point `Gamma0=10`
  (`max|A_T - A_U| = 0.0`; `chi_full` agrees to `0.0`) -- confirming this is
  one physical system probed along two different scaling paths.
- Path T (only the phonon/orbital channel scaled, `D_T` singular): measured
  `nu[R_S] = -0.002` (Class III, protected).
- Path U (all dissipation scaled uniformly, `D_U` invertible): measured
  `nu[R_S] = 1.961` (Class II).
- Class fan: sweeping the scaling direction `theta` from the singular
  endpoint (`theta=0`, `nu~0.003`) to the interior (full-rank `D(theta)`,
  `nu` plateauing near 1.93-1.96) shows the class flipping exactly where the
  rank of `D(theta)` changes -- a transition invisible to any framework that
  assigns one class per physical configuration rather than per scaling path.

## D4: preasymptotic masquerade vs the finite polynomial certificate

Overall: **PASS**

- Exact cancellation point (symbolic, rational arithmetic): at
  `v02* = -0.35`, the master moments are `[0, 0, 0, 895/2688]` identically --
  `m_2 = 0` exactly, confirmed symbolically, not by floating-point smallness.
- Detuning `v02` by `delta = 1e-5` off the exact cancellation point leaves a
  tiny nonzero `m_2`, producing a false plateau at `nu_eff ~ 4` spanning
  **2.43 decades** in `Gamma` before crossing over to the true asymptotic
  order.
- The finite polynomial certificate (`deg_Gamma Q - deg_Gamma N`, Theorem 8.1

  certificate to within `0.001` -- confirming that only the certificate is
  window-independent, while any finite-window experimental fit performed
  inside the false-plateau region would report the wrong class.
