## 1. 理論名と中心主張

**保護応答の熱力学**（Thermodynamics of Protected Response, **TPR**）。

一言で：*散逸応答の三分類（とくに Class III 保護）は熱力学的に成層する。
保護チャネルは「dark-state支持型＝無償」と「コヒーレンス維持型＝有償」に
排他的に二分され、有償保護には散逸スケールに比例するエントロピー生成の下界
——**保護料（protection premium）**——が課される。応答指数 ν と EP指数 μ の
対 (ν, μ) は結合fanをなし、その実現可能領域が平衡・非平衡・dark支持の
熱力学的クラスを、断層撮影なしの2スロープ測定で分離する証人になる。*

中心対象は結合指数対の実現可能領域：

$$\mathcal T^{\mathrm{eq}} \;\subsetneq\; \mathcal T^{\mathrm{dark}} \;\subsetneq\; \mathcal T^{\mathrm{neq}}
\qquad \subset \{(\nu_S(q),\, \mu_S(q))\}$$

（有限温度詳細釣合い／dark支持を許す非平衡／一般非平衡GKSL。いずれも宣言
熱力学界面＝自律化された浴・電池の組、killing型介入、固定source/readout、
宣言経路族 κ=κ₀Γ^q の下で定義する。）

### 1.1 事前撤回した素朴経路

**(i)「定常コヒーレンスは有償」——撤回。** 平衡Gibbs状態 ρ_β は H と可換だが、
測定基底や局所基底では非対角成分（コヒーレンス）を持ち、その維持に EP は要らない。
資源理論的コヒーレンス一般を価格の担い手にすると反例で即死する。正しい担い手は
**Γスケール散逸子の固有基底（不変代数）に対して非対角で、かつ定常電流を運ぶ成分**
——保護係数 R_{S,0} ≠ 0 を通すために不可欠な成分——に限定する（T2 の仮定 (P2)）。

**(ii) 半古典駆動の簿記——撤回。** external field を古典パラメータとして扱うと、
場が系に行う仕事が熱力学勘定の外に落ち、「EPゼロの有償保護」が見かけ上作れて
しまう。全主張は**自律化**（駆動を有限だが大きい電池／単色モードとして量子化し、
浴をすべて温度指定する）した界面で述べ、半古典簿記はその極限として整合することを
Gate P0 で先に固定する。P0 が通らない場合の縮退先は Stop 条件に明記（§6）。

### T2（決定点・条件付き候補）保護料下界

**仮定:**
- (P1) 宣言熱力学界面（自律化、Gate P0 で固定）の下の GKSL；
  Γスケール散逸子の固有基底（不変代数）𝔄 を宣言

  補題として証明する：R_{S,0} は P ブロックの応答であり、P が dark 支持で
  ない限り c_⊥ = 0 なら R_{S,0} = 0）

**主張:** σ_ss ≥ C·Γ·c₀² + O(1)。
**証明経路:** Γスケール位相緩和はコヒーレンス c_⊥ を速度 ∼ Γ|c_⊥|² で破壊する。

アフィニティを横切る。フラックス×アフィニティの標準下界（および必要なら
熱力学的不確定性型の補強）で σ ≥ C·Γ·c₀²。

総EPから「𝔄横断コヒーレンス電流」に差し替えて再定式化する（Stop条件）。

### T3（構造定理候補）二相性の完全性

**主張:** 宣言界面の下で Class III 保護チャネルは排他的に二分される：
- (a) **dark-state支持型（無償）**: 保護Riesz射影 P の像が全Γスケールjump作用素

- (b) **コヒーレンス維持型（有償）**: (a) でないすべて。T2 が適用され σ_ss ≳ Γ

**証明経路:** P像とjump作用素核の交わりの代数的二分（DFS/noiseless subsystem
の Knill 型構造定理を「保護射影」に相対化する）。(a)⇒無償は dark 部分での

系として「無償の保護は dark 支持型に限る」——DFS 理論の外にある主張——が出る。

## 5. Stop D 差別化（先行研究監査つき）

- **TUR（熱力学的不確定性関係）**: 電流の相対揺らぎ精度を EP で下から縛る。

  指数値の離散証人という利点も別系統。ただし T2 の証明でTUR型補強を使う
  可能性があるため**引用必須**。
- **Harada–Sasa 等式・FDT破れ**: 応答と相関のギャップを EP に等値する
  （Langevin系）。TPR は等式でなく no-go／下界の階層構造であり、対象は
  sector 切断つき GKSL の保護係数。接続は将来課題として言及に留める。

  TPR の価格は定常**レート**の Γ スケーリング則で、応答分類に紐づく。
  §1.1 の撤回によって「資源理論の再包装」批判は構造的に回避される
  （価格の担い手を電流搬送コヒーレンスに限定した時点で資源理論の外）。
- **DFS / noiseless subsystems（Knill–Laflamme–Viola系）**: 保護の代数的
  存在条件。TPR は存在でなく**価格**を分類し、T3 で「無償はdark支持に限る」
  を主張する。構造定理は供給源として引用必須。
- **クロックの熱力学（Erker系）・自律QECのコスト**: 「機能はエントロピーを
  要する」という精神が最近接。相違は (i) 対象が介入分解応答指数、

  (iii) 認証済み三分類・fan体系への接続。**最強競合として明示的比較節を
  設ける**。
- **量子吸収機械・反転なし利得（LWI）**: 有償保護の実例供給源。一般分類
  定理は存在しない。実例→定理の関係として引用。
- **Kubo/KMS 平衡応答理論**: 平衡FDTは応答の対称性を与えるが、sector切断・
  保護プラトー・指数分類を扱わない。T1 はKMS対称性の現代的no-go形。
- **計画19 RCT との分離（内部監査）**: RCT の軸は「古典Markov vs 量子GKSL」

  共有するのは SMRT 基盤と証人設計思想（離散指数・ドリフト頑健）のみ。
- 正直な記載: T2 の下界定数 C が系依存になる可能性、および μ の log 補正
  （§3 T4）は、査読で「不等式が空虚」批判を受け得る2大弱点である。前者は
  Gate P3 のタイト性検証、後者は宣言窓つき定義で先回りする。
