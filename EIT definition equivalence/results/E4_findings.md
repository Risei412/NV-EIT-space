# Gate E4 結果報告 — T6(構成的存在定理)

## 結論(先出し): 構成に成功。ただし想定より重要な副産物あり

戦略MDが要求していた「dark state なしの厳密な実軸 transfer zero」の構成は
**成功**しました。ただし、その過程で最初に試した「単純な 2g+3e(対角励起共鳴、
実効的に3準位が独立に減衰)」モデルでは**厳密に不可能**であることが判明し、
それがなぜかを説明する、当初のプランになかった**より一般的な no-go 定理**が
副産物として得られました。ユーザーの依頼どおり、この経緯を隠さず報告します。

---

## 1. 最初の試み(単純 2g+3e、対角 G)は厳密には失敗

`src/model_2g3e_zero.py`, `src/model_2g3e_zero_fast.py` で、励起状態を3つに
増やし Cauchy–Binet 展開(3項の和)による位相相殺を狙ったが、
系統的なグリッド+多点始動 Newton 探索(2112〜数千通りのパラメータ組)で
**厳密零点は一つも見つからなかった**。

### なぜ失敗したかの証明(一般化 no-go 定理)

対角励起共鳴 `G = diag(1/a_1,...,1/a_Ne)`(各 `a_j = γ_j + i x_j`, `γ_j > 0`
— 任意の受動的減衰、gain なし)のとき、

```
D†GD = Σ_j g_j v_j v_j†      (v_j = D の j 行目 ∈ C^2, g_j = 1/a_j)
```

は Hermitian PSD な dyad `v_j v_j†` を `Re(g_j) > 0` で重み付けた和である。
`w†(D†GD)w = 0` となる `w≠0` が存在すると仮定すると、実部を取って

```
Σ_j Re(g_j) |v_j†w|^2 = 0
```

各項が非負なので全項が0、すなわち全ての `v_j` が `w` と直交する
1次元部分空間に属する ⟹ `rank D ≤ 1`。これは `rank D = 2` の仮定と矛盾。

**⟹ 励起状態数 N_e に依らず、対角(コヒーレント結合のない)励起多様体では、
受動的な減衰である限り、dark state なし(rank D = 2)の厳密な実軸零点は
決して存在しない。**

200,000 回のランダム試行(N_e = 2〜5、ランダムな D・減衰率・離調)で
反例0件を確認し、この定理を数値的に検証済み(`model_2g3e_zero.py` 内の
検証コード)。これは戦略MDの Theorem IV(2g+2e no-go)を**任意次元**に
拡張する、より強い結果である。

---

## 2. 脱出経路: 励起状態間のコヒーレント結合(closed-loop)

上記証明の鍵は「G が対角」という仮定である。励起状態間に
**コヒーレントな(Hermitian, エネルギー保存的な)結合 J** を導入すると
(`G = (diag(a_j) + J)^{-1}` が非対角になる)、`D†GD` はもはや
正実部重み付き PSD dyad の和という形を取らず、上記の障害は消える。

`src/model_2g3e_closedloop.py` で e2–e3 間に実結合 `J23` を持つ 2g+3e モデルを
構成し、1500点のランダム多点始動 Newton 探索を行った結果、
**238 個の厳密解**(`rank D = 2`、機械精度で `χ_full(δ0) = 0`)が見つかった。

### 検証済み構成例(7条件チェックリストのうち6項目を確認)

```
δ0        = 1.5366   (実軸上の観測周波数)
Δ2        = -2.8571  (e2 の離調)
Δ3        =  2.5308  (e3 の離調)
J23       =  6.2474  (e2-e3 コヒーレント結合)
φ         =  0.5270  (制御双極子の e3 成分の位相)
```

| 条件 | 判定 |
|---|---|
| 1. χ_full(δ0) = 0(機械精度) | ✅ True |
| 2. rank D = 2(optical dark vector なし) | ✅ True |
| 3. 定常純 Lindblad dark state なし | ✅ True |
| 4. R_S(δ0) ≠ 0 かつ R_S = −χ_cut | ✅ True |
| 5. セクターカットで透明化が消える(χ_cut absorptive) | ✅ True |
| 6. ATS 非分裂(極分離/半値幅比) | ✅ 0.029(≪1、非分裂) |
| 7. control-induced(K_pc ≠ 0、交差結合が寄与) | ✅ True |

極の位置(δ 平面): `-0.163∓6.64i, -0.163+4.64i, 0.0-1.0i` —
実部がほぼ揃っており(|Δ Re| ≲ 0.16)、幅の異なる極同士の干渉として
現れる、ATS ダブレットではなく EIT 型の相殺構造。

## 3. 解釈と次のステップ

- **主結果(構成)**: dark state のない厳密 EIT 零点は 2g+2e の対角励起
  多様体では不可能だが、励起状態間のコヒーレント結合(closed-loop、
  例えば NV センターならスピン軌道・スピンスピン・歪みによる励起状態
  混成)があれば厳密に実現できる。これは戦略MD T6 の目標を達成する。
- **副産物(拡張 no-go)**: 対角励起共鳴という条件そのものが本質的障害
  であり、次元(2e か 3e か)は無関係。この一般定理は戦略MD Theorem IV
  の主張より強く、論文の理論的な柱を一つ追加できる
  (「Theorem IV': 任意次元・対角励起多様体の一般化 no-go」)。

  写像。次のセッションで T6 の定理文書化と `tex/equivalence_theorems.tex`
