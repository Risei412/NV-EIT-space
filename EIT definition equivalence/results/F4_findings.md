# Gate F4 結果 — 透明化フロア定理(閉じた形の解析公式)

## 結論: 探索ではなく理論決着。閉じた形の公式を導出・機械精度で検証済み

F4 preview(前回セッション)で、命題A(strict-accretive no-go)が
`γ_g>0` を含む全コヒーレンス空間に拡張されることを確認した。今回、
**標準的な(dark stateを持つ)Λ系**(2g+2eのdark-state-freeモデルとは別物、
probe と control が同一の励起状態 `e` に結合する教科書的トポロジー)に対して
**閉じた形の透明化コントラスト公式**を導出し、機械精度で検証した。

## 1. 導出

Matched 応答(source = readout = プローブ双極子)の 2×2 モデル

```
A(δ) = [[a1, iΩ_c/2], [iΩ_c/2, g]],   a1 = γ_e − iδ,   g = γ_g − iδ
```

（Hermitian control 結合、正しい規約は F0/F1 監査で確定した
`A_offdiag = i·H_offdiag` — model H の教訓をそのまま踏襲）で、
二光子共鳴点 `δ=0` における応答を直接計算すると:

```
χ_full(0) = γ_g / (γ_e·γ_g + β),     β = Ω_c²/4
χ_cut(0)  = 1/γ_e
```

したがって透明化コントラストは **厳密に閉じた形**で:

```
C_S(γ_g, Ω_c) = 1 − χ_full(0)/χ_cut(0) = β / (γ_e·γ_g + β)
             = Ω_c² / (4γ_e·γ_g + Ω_c²)
```

残留(フロア)吸収:

```
1 − C_S = 4γ_e·γ_g / (4γ_e·γ_g + Ω_c²)  ≈  4γ_e·γ_g/Ω_c²   (γ_g ≪ Ω_c²/4γ_e のとき)
```

これは標準的な EIT の残留吸収スケーリング(Fleischhauer–Imamoglu–Marangos
のレビューに載る教科書公式)そのものであり、**命題A(F2のstrict-accretive
no-go)の系として再導出**できたことになる。300個のランダムな
`(γ_g, Ω_c)`(`γ_g∈[10⁻⁶,10]`, `Ω_c∈[10⁻²,10]`)に対して、
数値計算と閉じた形の公式の差は **2.2×10⁻¹⁶(機械精度)**。

## 2. 途中の落とし穴(正直に報告)

最初、`δ=0` に固定せず「`Re(χ_full)` の全域最小値」を探索する検証を書いたところ、
`γ_g=0.135, Ω_c=3.84` で `C_S = -0.248`(コントラストが負、命題A違反に見える)
という結果が出た。詳しく調べると、**`Re(χ_full)` 自体は該当点(`δ=-6`)で
`0.0337`(正)であり、命題Aは破れていなかった** — 単に共鳴から大きく外れた
テイル領域で `χ_full` と `χ_cut` がともに小さくなる中、たまたま比が1を超えて
`C_S<0` に見えただけで、これは物理的におかしくない(共鳴から外れれば
「セクターが背景より悪い」こともある)。EIT コントラストは
**二光子共鳴点 `δ=0` でのみ意味を持つ量**であり、大域最小探索は
Autler-Townes 領域(`Ω_c` が大きい場合)のテイルを誤って拾ってしまう
不適切な検証方法だった。`δ=0` に固定した正しい検証で機械精度の一致を確認した。

## 3. 相図

`figures/F4_transparency_floor_phase_diagram.png` に
`(γ_g, Ω_c)` 平面上の残留吸収フロア `1-C_S` の対数カラーマップを生成した。
`γ_g→0` または `Ω_c→∞` で `1-C_S→0`(完全透明化への接近)、
`γ_g` が `Ω_c²/4γ_e` と同程度になると `1-C_S` が O(1) に戻る、という
標準的な EIT 相図が再現されている。

## 4. 解釈と F4 の決着

- **教科書の完全 EIT(`γ_g=0`)は、まさに `λ_min(Γ_full)=0` という
  passivity boundary そのもの**であり、`C_S(γ_g,Ω_c)` はその境界への
  接近速度を定量化する厳密な閉形式である。
- 元々の F4 の問い(「有限 `γ_g` で exact/absorption-only/finite dip の
  どれが起きるか」)は、**matched 応答では finite dip のみが可能で、
  そのコントラストの上限が上式で厳密に決まる**、という形で完全に決着した。
- 探索フェーズは不要だった。理論(命題A)から直接閉形式が出る、という
  F4 preview の予測どおりの結果。
