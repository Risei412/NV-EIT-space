# Gate F5-A 結果 — unmatched readout による受動的 escape の構成

## 結論: 成功。ただし「観測の定義」自体を拡張する必要がある

命題A(strict-accretive no-go)は **source と readout が同一ベクトル
(`p = c`, matched)** の場合にのみ成立する。`p ≠ c`(異なる検出モード/
直交成分を測る)にすると、この障害は数学的に必然的に消えることが
証明でき、かつ数値的に確認された。

## 1. なぜ unmatched で no-go が破れるか(機構)

Matched response `c†Gc` は(Re G = G†ΓG が正定値なので)常に
`Re(c†Gc) = c†(G†ΓG)c ≥ 0` という二次形式に拘束される — 符号が
固定されている。

Unmatched response `p†Gc`(`p≠c`)は二次形式ではなく**双線形形式**であり、
符号の制約が一切ない。50,000 回のランダム試行で確認: `Re(p†Gc) < 0`
となる割合は**約 49.7%**(符号はほぼ五分五分)。

したがって、2つの受動的(`Γ>0`)チャネル(または同一チャネルの2つの
射影)の unmatched な線形結合は、実部・虚部を共に厳密ゼロにできる
自由度を持つ。

## 2. 構成モデルと検証結果

`γ_g > 0` を含む matched-response floor 検証(F4)と同じ基底コヒーレンス
+励起多様体モデルを用い、**検出ベクトル `p` をプローブ双極子 `c` と
直交する方向に角度 `η` だけ傾けた**(`p = cos η · c + sin η · c_⊥ e^{iφ}`)。
すなわち、プローブと完全に同一の偏光/モードを測るのではなく、
**わずかに異なる偏光・モード成分を検出する**ことに相当する。

3000 回のランダム試行(`N_e=2`, ランダム Hermitian 励起混成、
ランダム `γ_g∈[10⁻³,1]`, ランダム制御結合)のうち **54 件**で
厳密な実軸零点が見つかった。

### 代表例(検証済み)

```
δ0        = -0.9842
γ_g       =  0.8233   (有限、ゼロでない)
Ω_c       =  0.2234
励起混成 J = 0.1715(e1-e2間 Hermitian結合)
η         =  1.443 rad(検出方向の傾き、readout ≠ probe)
```

| 条件 | 判定 |
|---|---|
| χ_full(δ0) = 0(機械精度) | ✅ `-1.1e-16 i` |
| **λ_min(Γ) = 0.7377 > 0**(厳密受動、gain なし) | ✅ |
| rank([d_p, 制御結合]) = 2(dark state なし) | ✅ |
| γ_g > 0(有限基底コヒーレンス減衰) | ✅ |
| control-induced(Ω_c → 0 で零点消失) | ✅ `χ(Ω_c=0) = 4.9e-5 - 3.9e-3 i ≠ 0` |
| 極分離 vs 半値幅 | 極: −2.989, 2.995, −0.0054(δ平面)、幅 ~1.0, 1.0, 0.82 — 分離/幅比 ≈ 3(境界的、ATSではないが要注意) |

## 3. 解釈 — Outcome III(戦略MD 8節)の確定

これは「物質内部のコヒーレント相殺」ではなく、**入出力ペア(源と検出器)
に付随する透明化**である。標準的な EIT 実験(透過率そのものを測る、
すなわち `p = c` の matched 配置)では命題A の no-go が厳密に効くため、
**この機構による厳密ゼロは通常の透過測定には現れない**。

これは失敗ではなく、EIT の定義そのものへの示唆である:

> **matched(透過率)な受動 Markovian 応答では、γ_g を含めても
> exact zero も absorption-only zero も存在しない(F4 の floor 定理)。
> 厳密な dark-state-free EIT zero を受動系のまま得るには、
> 検出方式を probe transmission から異なるモード/偏光成分の
> heterodyne 的観測へ一般化する必要がある。**

## 4. 次のステップ

- 極分離/半値幅比が境界的(~3)なので、パラメータを追加探索して
  ATS 除外がより明確な例(比 < 1)を選定する。
- この unmatched readout が物理的にどのような検出スキームに対応するか
  (偏光分離、ホモダイン、干渉計出力)を具体化し、実験的意味を持たせる。
- F5-B(完全 GKSL、ρ_0 のドレスによる自然な unmatch)は次のセッションで
  実施し、この F5-A の「人工的な検出器回転」なしに同じ機構が
  物理系内部から出てくるかを検証する。
- 戦略MD の元の柱(4)を「dark state なしでも通常の透過測定で厳密EIT」
  から「dark state なしでも**適切な観測ペアの下で**厳密EIT」へ
  正確に修正する。
