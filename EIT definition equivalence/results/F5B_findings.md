# Gate F5-B 結果 — 完全GKSL(2g+3e、ポピュレーション込み)

## 0. まず報告すべきこと: 実装バグを発見・修正した

`src/full_gksl_2g3e.py` の初版には**2つの符号バグ**があった。

1. **ベクトル化 Liouvillian の kron 引数順序が逆**だった
   (`kron(H,I)-kron(I,H.T)` は正しくは
   `kron(I,H)-kron(H.T,I)` — Fortran(列優先)vec の恒等式
   `vec(AXB)=(Bᵀ⊗A)vec(X)` を取り違えていた)。
2. **線形応答方程式の符号**も誤っていた
   (`(L0-iδ)drho=-i·comm` ではなく、正しくは
   `(L0+iδ)drho=+i·comm` — アンザッツ `ρ₁(t)=drho·e^{-iδt}` を
   運動方程式に代入して符号を再導出する必要があった)。

この2つのバグにより、**制御場なし・γ_relax込みの純粋な2準位吸収**という
最も単純なケースで `min Re(χ_matched) = -0.023`(負、gain のような見た目)
という誤った結果が出ていた。孤立2準位系での解析解 `χ=1/a₁` との比較で
発見し、修正後は機械精度(`1.4e-15`)で解析解と一致することを確認した。
**この誤りに基づいて「フル GKSL で matched 応答が負になる」と報告する
一歩手前だった** — ユーザーへの正直な報告として明記する。

## 1. 確認1: matched response の floor は完全GKSL(ポピュレーション込み)でも成立

修正後、制御場でドレスされた定常状態(`ρ_ee>0` の複数励起状態、
基底コヒーレンス、`γ_g>0`)を持つ完全な密度行列計算で、
**matched 応答(プローブと同一の e1-g1 コヒーレンスを読み出す)** を
150 通りのランダムパラメータ(3励起状態、ランダム Hermitian 混成、
ランダム分岐比、`γ_g∈[10⁻³,10⁻¹]`, `γ_relax∈[10⁻³,10⁻¹·⁵]`)で走査:

```
min Re(χ_matched) over 150 trials = 0.0029   (>0、反例なし)
```

**F4 のフロア定理(命題A)は、コヒーレンスブロックの縮約モデルだけでなく、
ポピュレーション動力学を完全に含む GKSL 密度行列計算でも成立する。**
縮約時に捨てた「ポピュレーション経路」が符号制約を破る余地はなかった。

## 2. 確認2: unmatched(Raman 基底コヒーレンス)読み出しでは厳密零点が自然に存在

読み出しを `⟨e1|ρ|g1⟩`(matched)ではなく `⟨g2|ρ|g1⟩`
(**基底コヒーレンスの Raman 読み出し** — Raman heterodyne 検出や
コヒーレント Raman 散乱として実験的に確立された標準的な検出方式)
に変えると、60 回のランダム試行のうち **3件の真の厳密零点**
(機械精度、パラメータ有限)が2パラメータ(δ, 第3双極子成分の位相)の
fsolve で見つかった(残り3件はパラメータ発散の逃走解)。

代表例の検証:

```
δ0 = -0.9468
rank([d_p, d_c]) = 2          (dark state なし)
GKSL 散逸率はすべて構成上 ≥0   (物理的に受動、gain なし by construction)
γ_g = 0.00325 > 0, γ_relax = 0.00117 > 0 (有限)
χ_raman(δ0) = 1.1e-15 - 4.0e-16 i  (厳密ゼロ)
χ_matched(δ0) = 0.639 - 0.157 i    (同じ点で matched 応答は非ゼロ・正の吸収)
```

これは F5-A(人為的に検出方向を回転させた構成)と同じ結論を、
**検出器を回転させる操作なしに、標準的な代替観測量(Raman コヒーレンス)
だけで**、完全な物理 GKSL モデルの内部から確認できたことを意味する。

## 3. 未解決の注意点(正直に報告)

**control-induced 条件(戦略MDの Condition 5/7)がこの Raman チャンネルには
そのままでは適用できない。** `Ω_c=0` にすると `χ_raman(δ)` は
**全ての δ で恒等的にゼロ**になる(この観測量自体が制御場の結合 `d_c`
を経由してのみ存在するため)。つまり「制御場がなければ非ゼロだったものが
制御場でゼロになる」という意味での cancellation ではなく、
「観測量自体が制御場なしには存在しない」という構造的な自明性であり、
標準的な EIT の dark resonance が `Ω_c→0` で滑らかに消える現象と類似ではあるが、
戦略MDの条件5の文言をそのまま適用するには再定式化が必要。

## 4. 結論と次段階

- **F5-B は「完全 GKSL で unmatch が自然に生じるか」という問いに Yes で答えた。**
  人為的な検出器回転（F5-A）なしに、標準的な代替観測量（Raman コヒーレンス）
  だけで、受動・dark-state なし・有限 γ_g の厳密零点が得られる。
- 主結果の分岐は最終的に **Outcome III で確定**と見てよい: 通常の透過測定
  (matched)では厳密 EIT zero は不可能(F4 フロア定理、コヒーレンス
  ブロックでも完全 GKSL でも同じ)。dark-state-free な厳密 EIT zero は、
  **入出力ペアを適切に選んだ(Raman 検出などの)観測でのみ**受動・gain なしで
  実現できる。
- 次にやるべきこと: (a) Condition 5/7 を「観測量が制御場に依存して初めて
  定義される場合」に適切に再定式化する、(b) 極分離/半値幅比の確認
  (ATS 除外)をこの Raman zero に対しても行う、(c) NV センターにおける
  Raman heterodyne 検出との対応を具体化する。
